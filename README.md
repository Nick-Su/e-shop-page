# E-Shop Page
Basic template page for an e-shop

## Tech stack
* React v. 17.0.2
* Webpack v. 5.61.0
* Swiper v. 6.5.1

## Installation
Clone the project
``` git@github.com:Nick-Su/e-shop-page.git ```

Install the dependencies
``` yarn install ```

Start a local server
``` yarn start ```

Visit page
http://localhost:3000/
